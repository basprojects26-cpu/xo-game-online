# Tic-Tac-Toe Ultimate

This project is a multiplayer Tic-Tac-Toe web application.

## Structure
- `public/index.html`: Main HTML file for the client UI.
- `public/style.css`: Styles for the client UI.
- `public/script.js`: Client-side JavaScript containing the game logic and Socket.io client setup.
- `server.js`: Node.js server using Express and Socket.io for multiplayer synchronization.
- `xo-game-online.html`: Another version/backup of the HTML file.

## Goal
A real-time competitive experience with persistent profiles and social connectivity.
